# tpx-pkg-docker
Three Pixels package to install docker.

## How to use (CentOS 7)

1. Clone repository:

```bash
$ git clone https://github.com/valmorsecco/tpx-pkg-docker.git
```

2. Go to tpx-pkg-docker folder:

```bash
$ cd tpx-pkg-docker
```

3. Run install.sh:

```bash
$ chmod +x install.sh
$ ./install.sh
```
